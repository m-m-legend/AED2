# 📊 AED2 - Algorithms and Data Structures II

A repository dedicated to storing **advanced algorithms**, **complex data structures** and **academic projects**, implemented in **C++ and Python**.

## 📋 Description

This repository contains implementations of more advanced algorithms and sophisticated data structures learned during the Algorithms and Data Structures II course at UERJ. It includes resolution of practical exercises, three academic test projects that explore different paradigms (Divide and Conquer, Dynamic Programming and Specialized Structures) and their applications to classic problems.

## 🚀 Content

### 🧮 Advanced type of Algorithms

- **Dynamic Programming**: Memoization, Tabulation
- **Divide and Conquer**: Merge Sort, Quick Sort, Kadane's algorithm
- **Greedy Algorithms**: Optimization problems
- **Brute Force**: Simple solving of common problems

### 🏆 Academic Projects

#### 1️⃣ **Project 1: Kadane's Algorithm with Divide and Conquer**
- Problem: Maximum subarray sum
- Technique: Divide and Conquer
- Complexity: O(n log n)
- Application: Linear optimization problems

#### 2️⃣ **Project 2: SCML with Dynamic Programming**
- Problem: [SCML - Specify Complexity]
- Technique: Dynamic Programming
- Complexity: O(n²) or optimized
- Application: Sequence and subsequence problems

#### 3️⃣ **Project 3: Infinite Wall with Circular Linked List**
- Problem: Cyclic resource/structure management
- Structure: Circular Linked List
- Complexity: O(1) for cyclic operations
- Application: Structures with cyclic behavior

## 📖 How to Use

1. **Clone the repository**:
   ```bash
   git clone https://github.com/m-m-legend/AED2.git
   cd AED2
   ```

2. **Compile a specific file**:
   ```bash
   g++ -std=c++11 -O2 -o program file.cpp
   ```

3. **Run the program**:
   ```bash
   ./program
   ```
